# UDP Server API Docs

### Remote UDP Server Configure

```toml
[UdpServer]
protocol = "UDP"
ip = "139.199.5.24"
port = 2333
```

### type definition

* <host>
```json
{
    "ip": "<string>",
    "port": "<number>"
}
```

### Peers --[SEND]--> Server

- `method`
  - `'SEEK'` ：创建 / 加入聊天室
  - `'UPDATE'` ：选举了新 leader
  - `'LEAVE'` ： 聊天室最后一位退出聊天室

```json
{
    "method": "<string>",
    "data": {
        "name": "<string>",
        "token": "<string>",
        "public": "<host>",
        "inner": "<host>"
    }
}
```
